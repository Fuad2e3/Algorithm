# Algorithm Implementations in Java

This repository contains various algorithm implementations and data structures written in Java. The project serves as a collection of solutions to classic algorithmic problems, focusing on efficiency and clarity.

## Project Structure

The project currently includes the following implementations:

### 1. Crab Graph ([CrabGraph.java](file:///C:/Users/fuadk/Documents/GitHub/Algorithm/CrabGraph.java))
An implementation of a "Crab Graph" data structure. This specialized graph allows designating specific vertices as "body" nodes, which have a configurable limit on the number of "leg" edges they can support.

### 2. Insertion Sort Shift Counter ([pb2.java](file:///C:/Users/fuadk/Documents/GitHub/Algorithm/pb2.java))
An extension of the standard Insertion Sort algorithm that not only sorts an array but also calculates the total number of shifts performed during the sorting process. This is often used to measure the "inversion count" or the degree to which an array is unsorted.

### 3. Ice Cream Parlor / Two Sum ([pb3.java](file:///C:/Users/fuadk/Documents/GitHub/Algorithm/pb3.java))
A solution to the "Ice Cream Parlor" problem, which is a variation of the classic Two Sum algorithm. It identifies the indices of two elements in an array that sum up to a specific target value.

## Getting Started

### Prerequisites
- Java Development Kit (JDK) 8 or higher.

### Compilation
To compile any of the Java files, use the `javac` compiler from your terminal:

```bash
javac FileName.java
```

### Execution
Once compiled, you can run the program using the `java` command:

```bash
java FileName
```

#### Example:
```bash
# Compile the CrabGraph implementation
javac CrabGraph.java

# Run the compiled class
java CrabGraph
```
